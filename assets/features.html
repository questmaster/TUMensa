Main Screen

  • Display - Here you see the meals available at the different counters of the selected Mensa and day. If enabled, the result of user votes is displayed below each meal.
  • Navigation - To navigate through day you can click on the arrows on top or use swipe gestures, if enabled. To get back to today you can select menu > today or use the double tap gesture if enabled.
  • Data update - To update, press menu > update. If you have enabled auto updates this should not be necessary.
  • Context Menu - Long pressing on a meal or the mensa location on top opens a context menu. The latter menu shows a selection dialog for the mensa location. The context menu for a meal can show two entries. First a share with friends, to share a meal on Twitter, email and so on, and secondly a vote dialog (if enabled and available for selected meal), where you can rate three aspects of your meal.

Settings

  • Mensa location - Here you can select the mensa you go to.
  • Delete old menu entries - if enabled, only data for this and next week is stored. Previous entries are deleted. (With an App update old menu data can also get deleted, regardless of the selection here).
  • Auto update - Automatically updates meal data, if needed. This is recommended, if you have a permanent internet connection.
  • Enable voting - With this enabled, you can rate different aspects of your meal. This option is only recommended if you have a permanent internet connection, as sending and updating is done automatically in background!
  • Enable gestures - Enables swipe and double tap gestures for navigation, see above.
  • Select theme - You can select a dark or a white theme.
